# 算法竞赛

算法竞赛入门 · ACMer 起步指南

📅 2026-03-23 · 🕒 12分钟阅读

很多刚接触算法竞赛的同学都会问：“我是零基础，怎么开始？” 这篇文章写给准备入坑 ACM/ICPC、蓝桥杯、CSP 等算法竞赛的萌新，帮你理清思路，少走弯路。

一、心态准备：竞赛不是速成

算法竞赛考察的是 算法思维 + 代码实现能力 + 比赛经验，不是背模板就能拿奖的。前期可能会很痛苦，一道题调一整天很正常。但坚持下来，你会发现自己的逻辑思维和代码能力会有质的飞跃。

“AC 不是终点，WA 也不是终点，放弃才是。” —— 某退役 ACMer

二、语言选择：C++ 是主流

国内算法竞赛（ICPC、蓝桥杯、CCPC）默认语言是 C++，因为 STL 库提供了 vector、set、map、sort 等高效工具。如果你只会 Python，也可以参加，但部分 OJ 对 Python 的时间限制比较严格。

  • C++：推荐，掌握 STL 基本够用
  • Python：适合入门练思维，但比赛可能吃亏，因为运行效率太低了，同样的逻辑可能就会超时
  • Java：也可以，但代码量偏大 （真的有一个你们的学长是拿Java写算法题的）

三、OJ 平台推荐（刷题网站）

  • 洛谷 (luogu.com.cn) — 国内最友好的 OJ，题解丰富，适合新手
  • Leetcode (leetcode.com) — 类型丰富，题单覆盖面广，推荐 '灵茶山艾府' 的入门题单
  • 牛客网 (nowcoder.com) — 国内校招刷题，有竞赛模拟
  • POJ / HDUOJ — 老牌 OJ，经典题多

四、刷题路线（按难度递进）

不要一开始就啃难题，容易劝退。推荐顺序：

  1. 语法基础 — 循环、数组、字符串、函数
  2. 简单算法 — 枚举、模拟、贪心、排序
  3. 数据结构 — 栈、队列、链表、并查集、哈希表
  4. 搜索 — DFS、BFS、回溯
  5. 图论 — 最短路、最小生成树、拓扑排序
  6. 动态规划 (DP) — 背包、线性 DP、区间 DP
  7. 数论/数学 — 素数筛、GCD、快速幂
  8. 高级专题 — 线段树、网络流、字符串匹配

五、推荐

  • 《算法竞赛入门经典》（紫书）— 刘汝佳 著，经典教材，适合系统学习
  • 《算法竞赛进阶指南》（蓝书）— 李煜东 著，进阶必备
  • 《挑战程序设计竞赛》（白书）— 日本选手写的，代码简洁
  • 《算法导论》 — 当字典查，不推荐新手通读

六、关于比赛

  • Leetcode 周赛 — 频率高，不定期大厂内推
  • Codeforces — 全球最活跃的竞赛平台 ＆ ACMer的训练赛平台
  • 牛客小白月赛 — 每月一场，难度友好

七、学习资源

  • 洛谷题单 — 按知识点分类，跟刷就行
  • OI Wiki (oi-wiki.org) — 中文算法竞赛百科全书，免费且全面
  • B站/YouTube — 搜索“算法竞赛入门”，很多高质量视频
  • GitHub 算法模板 — 收藏几个常用模板，方便复用

八、给新手的几点建议

  • ✅ 先刷 100 道简单题，培养代码手感
  • ✅ 不要只背代码，要理解思路，多写注释
  • ✅ 学会看题解，但看完要自己重新 AC 一遍
  • ✅ 参加一场 Codeforces 比赛，感受真实赛制
  • ✅ 找队友一起刷题，交流讨论效率更高
  • ✅ 写博客记录错题，建立自己的错题本

九、常见误区

  • ❌ 一上来就刷难题 → 容易放弃
  • ❌ 只看不写 → 眼高手低
  • ❌ 盲目追求题数 → 不如把一道题吃透
  • ❌ 忽略比赛经验 → 平时刷题和比赛节奏完全不同

算法竞赛是一条漫长但值得的路。无论你是为了升学、找工作，还是单纯热爱编程，坚持下去，终会看到自己的成长。

祝你 AC 如风，早日成为大佬！

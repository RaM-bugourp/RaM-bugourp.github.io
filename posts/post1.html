# Django

Django Vue AdminX 框架开发实战

📅 2026-03-22 · 🕒 8分钟阅读

一套完整的企业级后台管理系统框架，基于 Django 5.2 和 Vue 3 开发。

核心功能

  • RBAC 权限控制（菜单、按钮、数据三级权限）
  • 代码生成器：一键生成前后端 CRUD 代码
  • 操作日志：中间件自动记录所有 API 请求
  • 任务调度：基于 APScheduler 的定时任务
  • 系统监控：CPU、内存、磁盘实时监控

上手框架

git clone https://github.com/niezhicheng/django-vue-adminx
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

这套框架已经帮助我快速交付了多个企业后台项目，大大提升了开发效率。

框架并非本人开发，但是超高的完成度实在让我忍不住想分享完整源码

原框架：Django Vue AdminX
